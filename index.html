Lightweight PHP framework with elegant architecture for building maintainable applications

Install via Composer:

composer create-project -n -sdev --prefer-dist no-global-state/krystal-skeleton ./

Why use Krystal?

Here are some pros

Easy to learn

To get started, all you need to configure is database connection and routes. And then simply you can start writing controllers and services!

Lightweight

The framework's size itself is around 1MB. But at the same time, it contains almost everything you need to develop modern PHP applications quickly and without a pain.

Fast

Krystal doesn't load a lot of classes every request, which makes it pretty fast.
